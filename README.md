# 🏆 Ranked Queue System (Java)

A Java application that simulates a ranked matchmaking system, calculating player performance based on wins and losses and assigning competitive ranks.

---

## ⚡ Overview

This project evaluates heroes (players) by calculating their win balance and assigning them a rank tier based on their performance.

---

## 🎮 Ranking System

| Wins        | Rank       |
|------------|------------|
| < 10       | Iron       |
| 10 - 20    | Bronze     |
| 21 - 50    | Silver     |
| 51 - 80    | Gold       |
| 81 - 90    | Diamond    |
| 91 - 100   | Legendary  |
| 101+       | Immortal   |

---

## 🚀 Features

- 📊 Win/Loss balance calculation  
- 🏅 Automatic rank classification  
- 🔁 Multiple heroes processing  
- 🧠 Clean and readable logic  

---

## 🧩 Concepts Applied
